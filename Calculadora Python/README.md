El objetivo del proyecto es la creación de una Calculadora básica en lenguaje python para la práctica y familiarización del mismo lenguaje.
Se propuso trabajar con las siguientes especificaciones:
Lenguaje python
Operaciones/Funcionalidades soportadas inicialmente en versión 1.0 : 
- Suma
- Resta
- Multiplicación
- División
- Operaciones en negativo
- Operaciones compuestas (> 2 términos)

Planes a futuro:
- Incluir la posibilidad de graficar en base a una función (ej:f(x2 +2x) * 2y  3xy2 =3 )  
- UI de color ajustable en menu desplegable de configuracion.
- Registro local de hasta 10 cuentas previas con la posibilidad de cargar las funciones utilizadas o el resultado de las mismas.

Para ejecutar la Calculadora nos dirijimos al proyecto "Calculadora Python" en la terminal y ejecutamos el archivo main, deberia quedar asi " cd ruta a la carpeta/Calculadora Python" y ejecutar "python main.py"
