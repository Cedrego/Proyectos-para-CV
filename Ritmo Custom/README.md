Lenguajes: HTML,CSS y JavaScript
Modo de ejecuccion: Urilizamos "Live Servver"(Extension en Visual)

Objetivo
Diseñar y crear una web que sirva como medio de visualización del ritmo de una canción en reproducción con Javascript, el cual introduciendo una canción crea una representación visual del ritmo de una canción en base a la frecuencia.
Se propuso trabajar con las siguientes especificaciones:
- Lenguaje JavaScript, HTML y CSS.
- Funcionalidades soportadas para la versión 1.0:
    - Ingresar musica descarga tipo MP3.
    - Posibilidad de cambiar entre estilos de Diseño de ritmo.
    - Posibilidad de cambiar entre tipos de colores, multicolor o por intensidad.
    - Tener todos los botones para interactuar con la música.
